About Shughran

Shughran is a comprehensive organizational management system built on CodeIgniter 3.x framework. It provides multi-department structure management, branch administration, user role management, and comprehensive reporting capabilities.

Key Features:

  • Multi-level organizational hierarchy management
  • Role-based user access control
  • Branch and sub-branch administration
  • Comprehensive reporting system
  • Multi-language support (Bangla and English)
